# **Skillorbit Student Performance Prediction Dashboard**
This is a streamlit application which is able to predict your grade class and GPA with academic and racial information.
## **Technical Info of The Model Used**
A RandomTreeRegressor from the scikit-learn library in python was used to predict the GPA which inturn falls in a grade class.
The Kaggle dataset used: [Student-Performance-Dataset](https://www.kaggle.com/datasets/rabieelkharoua/students-performance-dataset).\
The link to access the streamlit deployed application : [Dashboard](https://skillorbitstudentperformancepredictiondashboard-yzlq6pf7iexhst.streamlit.app/).\
